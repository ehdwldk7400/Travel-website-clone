# 구현 기능
## 공통
- Beautifulsoup, Selenium을 이용한 실제 데이터 수집
- Bcrypt를 사용하여 패스워드 해싱
- JWT를 사용하여 로그인 시 토큰 발행 
- 회원 가입 및 로그인
- 이메일 

## 상품
- 메인페이지
  - 메인카테고리 / 서브카테고리 상품 분류 구현 
- 상세페이지 
- 검새 기능 

## 리뷰
